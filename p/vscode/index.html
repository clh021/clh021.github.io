前言

VSCode的快捷键继承了一些IDE风格，有VS的身影，也有Emacs的身影。

快捷键组合方式

  1. Ctrl + Shift + ? : 常规按键组合
  2. Ctrl + V Ctrl +V : 同时依赖一个按键的组合
  3. Shift + V c : 先组合后单键的输入
  4. Ctrl + Click: 键盘 + 鼠标点击
  5. Ctrl + DragMouse : 键盘 + 鼠标拖动

标签

快捷键 作用
Ctrl + tab,alt + 在标签之间切换
Ctrl+(number of tab groups open) 创建一个新的标签组
Ctrl+(n) 在标签组之间切换
Ctrl + alt + left/right arrow key 将当前选项卡移至左/右选项卡组
Ctrl + w 关闭当前标签
Ctrl + F4, Ctrl + W 关闭编辑器
Ctrl + Shift + PgUp/PgDown 切换标签页的位置

终端

快捷键 作用
