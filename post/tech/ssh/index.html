ssh

2022-04-29 08:04:17
阅读时长: 1 分钟

Use 

ssh -L 4333:localhost:8000 root@xxxxx -N -f
# 将本地的4333端口转发到远程的8000端口,这意味着你在本地调用localhost:4333即可以获取到xxxxx服务器上8000端口的结果.

ssh -L 3333:prod:3306 root@jump -N -f
# 所有请求本地3333端口的请求,都会被转发为jump机器上对prod:3306的请求

ssh -R 8092:localhost:6666 root@xxxxx -N -f
# 将对xxxxx服务器8092端口的访问转发到本地的6666端口上,现在你可以让请求API的人直接请求xxxxxx:8092
