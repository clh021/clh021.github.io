ISO convert to Docker image

2022-04-24 23:50:47
阅读时长: 1 分钟

下载好 ISO 文件

wget -c http://pkg.loongnix.cn/loongnix/isos/Loongnix-20.1.livecd.mate.loongarch64.iso

准备文件和目录 

apt install squashfs-tools
mkdir unsquashfs
// 解压 出 ISO 其中的 filesystem.squashfs 文件，你的系统中还可能叫 squashfs.img
sudo unsquashfs -f -d unsquashfs/ squashfs.img

打镜像 

sudo tar -C unsquashfs -c . | docker import - loongnix-20.1

这种方式打出来的镜像太大，我觉得不可取。
