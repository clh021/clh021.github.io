备用命令

adb devices
adb reboot recovery

# 确保手机连接上了
fastboot devices
# 刷入 Recovery
fastboot flash recovery recovery.img
# 进入 Recovery 环境刷入系统，进入 ADB Sideload 模式
adb sideload lineageos.zip
# 刷入 GApps
adb sideload GApps.zip
# 刷入 Magisk
adb sideload Magisk.zip
# 刷入打过补丁的 boot_patch # 如果有的话，没有就是安装app启动app打补丁后得到补丁镜像再来操作
# 也有的是使用 recovery 子命令刷入 recovery ，主要看官网操作最后打补丁是哪个镜像
fastboot flash boot boot_patch.img

验证 Magisk 是否正确运行加载

看启动后，能否在第一个卡片中，正常显示当前版本号。如果可以，应该同时可以在设置中，看到 Magisk 设置类，打开 Zygisk 开关。打开后，提示重启，重启后能够在启动后的第一个卡片中看到 Zygisk 也显示了是。

安装 LSPosed 模块

拷贝 LSPosed 版本到后记，在 Magisk 底部模块导航菜单中，并选择从本地安装，选择拷贝好的zip包。

启动系统 USB 有线连接电脑拷贝以下包，并依次安装

  • F-Droid.apk
  • Magisk.apk
  • 应用宝.apk
  • LSPosed-v1.9.1-6990-zygisk-release.zip
  • Shamiko-v0.7.3-174-release.zip
