小米 6 使用 lineageOS 18.1 和 GApps

有一台闲置的小米 6，拿来装了第三方 ROM lineageOS，配合 Google 套件使用。 小米设备ROM下载站 小米6的ROM下载地址 小米10青春版ROM下载地址 MIUI历史版本下载合集

前期准备工作

打开开发者模式

设置->我的设备->全部参数，连续点击”MIUI 版本“

打开 usb 调试和设备解锁

设置->更多设置->开发者选项

  • 打开 usb 调试
  • 点击“设备解锁状态”，输入账号密码即可。如果需要等待 24*7,168 个小时，建议等待解 BL 锁

刷机过程

  1. 按住音量-+关机键进入 fastboot 模式 fastboot devices 查看设备是否正确连上
  2. 终端输入：fastboot flash recovery ./twrp-3.5.2_9-0-sagit.img
  3. 刷入成功后按住音量++关机键进入 TWRP
  4. 首次刷入会提示更改语言，选择中文
  5. 进入 TWRP，选择高级，选择 ADB Sideload
  6. 输入命令：adb sideload ./fw_sagit_miui_MI6_20.6.18_3f5b90d469_9.0.zip
  7. 观察设备，会提示刷入成功，桌面也会提示 Success
  8. 输入命令：adb sideload ./lineage-18.1-20210722-nightly-sagit-signed.zip
  9. 观察设备，会提示刷入成功，桌面也会提示 Success
  10. 继续刷入 GApps：adb sideload open_gapps-arm64-11.0-nano-20210730.zip
  11. 全部安装完成后，重启手机，进行一系列设置就能开心使用了。

也可以在 twrp 状态，将包全部拷贝进手机，一次性选好三个包，直接刷入，比起 adb 速度相当快。

安装 Magisk

开机状态输入命令 adb reboot recovery，进入 TWRP adb push Magisk-v23.0.zip /sdcard/Download/，找到对应路径安装 重启后开机 adb install Magisk-v23.0.apk 安装 Magisk 应用 下载地址：

其它

How to install adb on Linux? 

Debian / Ubuntu
`sudo apt install android-tools-adb`
Centos / Fedora / openSUSE
`sudo dnf install android-tools`
ArchLinux 及其衍生产品
`sudo pacman -Sy android-tools`

How to use adb? 

`adb devices` 检查设备是否正确连接到计算机。
`adb push fichero ruta` 将文件或文件夹从计算机发送到智能手机
`adb pull archivo` 将文件或文件夹从智能手机到电脑。
`adb install app.apk` 此命令将在智能手机上安装应用程序
`adb uninstall` 使用这个卸载应用程序。
`adb shell` 加载一个终端来控制我们的设备。
`adb reboot` 重新启动计算机。
`adb reboot-bootloader` 以快速启动模式重新启动
`adb reboot recovery` 使用此命令命令以恢复模式重新启动
